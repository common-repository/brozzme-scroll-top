== Description ==
**Brozzme Scroll Top**, is a lightweight plugin that adds a fully customizable scroll to top button on every page. The button appears when a person scrolls, disappears on top.
No coding skills require, No need to add any html to your theme.

** Works with Elementor page builder **
** Automatic Divi back-to-top disabling to use the plugin (and a customized button, without any css code !) **
** Automatic OceanWP back-to-top disabling to use the plugin **

Smooth scrolling animation, powered by jQuery
Customizable colors, Material design style, responsive.

1. Enable Brozzme Scroll Top, when enable, check for a Divi theme and then automatically disable the theme back-to-top button
2. Button background color
3. Button background color on hover
4. Button border-radius
5. Select floating (left or right)
6. Set height and width of the button.
7. Set padding from bottom and left or right.
8. Animation speed.
9. Animation type.

Languages : English (en_EN) and French (fr_FR)

Read more in the plugin Help tab.

Link to [Brozzme](https://brozzme.com/ "Brozzme") and [WPServeur](https://www.wpserveur.net/?refwps=221 "WPServeur WordPress Hosting").


== Installation ==
1. Upload Brozzme Scroll Top to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Enable and configure settings options in the Scroll Top in the admin panel.
